We are excited to announce the launch of the **Media Suite Community** site — a new home on the web for everyone who works with, builds on, or is simply curious about the [CLARIAH Media Suite](https://mediasuite.clariah.nl/).

## What is this site?

The Media Suite Community site is a Jekyll-powered, GitHub Pages-hosted hub designed to complement the official Media Suite documentation and tooling. It provides:

- **News and announcements** — release notes, research highlights, and community updates
- **Events** — workshops, webinars, and community calls, with registration links
- **Community information** — working groups, how to contribute, mailing list details
- **A platform for guest posts** — share your data story, tutorial, or research result with the broader community

The source code lives at [github.com/beeldengeluid/media-suite-community](https://github.com/beeldengeluid/media-suite-community) and contributions are welcome via pull request.

## Why a community site?

The Media Suite has grown steadily since its first public release, and so has its user base. Researchers in media studies, oral history, linguistics, film studies, and many other disciplines now rely on the infrastructure. Developers are building new tools on top of the API. Data curators are aligning collections with IIIF and linked-data standards.

Until now, community activity has been scattered across mailing lists, GitHub issues, and informal channels. This site aims to be a single, friendly entry point — low on jargon, open to all skill levels, and easy to contribute to.

## What's coming

Over the coming weeks we will be publishing:

- A guide to getting started with the Media Suite API in Python
- Highlights from recent community calls
- Announcements for the spring workshop series
- A curated list of published Data Stories from the community

## Get involved

The best way to shape this space is to participate:

1. **Follow the RSS feed** (link in the footer) to stay updated
2. **Join GitHub Discussions** to ask questions and share ideas
3. **Submit a guest post** by opening a pull request against the repository
4. **Come to a community call** — details posted monthly in Discussions

We look forward to building this together. If you have questions or suggestions, open an issue on GitHub or reach us at `mediasuite@beeldengeluid.nl`.

Welcome aboard!

— The Media Suite Team
