Join us for the **Media Suite Spring Workshop 2026**, a free full-day online event for researchers and developers who want to get more out of the CLARIAH Media Suite.

## Programme

| Time (CET) | Session |
|------------|---------|
| 10:00–10:30 | Welcome & infrastructure overview |
| 10:30–12:00 | Hands-on: Search, Compare, and Inspect tools |
| 12:00–13:00 | Lunch break |
| 13:00–14:30 | Working with ASR transcripts and metadata |
| 14:30–14:45 | Break |
| 14:45–16:15 | Introduction to the Media Suite API & Jupyter notebooks |
| 16:15–17:00 | Q&A, community discussion, next steps |

## Who should attend?

This workshop is designed for:

- **Researchers** new to the Media Suite who want a structured introduction
- **Experienced users** looking to deepen their API and scripting skills
- **Educators** considering the Media Suite for course assignments
- **Developers** interested in building on the infrastructure

No prior programming experience is required for the morning sessions. The afternoon API session assumes basic familiarity with Python.

## What you need

- A free Media Suite account (register at [mediasuite.clariah.nl](https://mediasuite.clariah.nl/))
- A laptop with a modern browser
- For the afternoon session: a working Python 3 environment (Anaconda or similar recommended)
